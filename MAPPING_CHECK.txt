=== ПРОВЕРКА СООТВЕТСТВИЯ НАЗВАНИЙ КОМПОНЕНТОВ И CSV ===

### 1️⃣ АЭРОГРАФИЯ (AerographySelect)

КОМПОНЕНТ:
- Тонировка
- Переход нескольких цветов
- Рисунок (персонаж, предмет, здание и т.д.)

CSV (текущие ключи):
- Аэрография тонировка
- Аэрография переход нескольких цветов
- Аэрография объемный рисунок

BACKEND маппинг (текущий):
  'tint' → 'Аэрография тонировка' ✓
  'gradient' → 'Аэрография переход нескольких цветов' ✓
  'volumetric' → 'Аэрография объемный рисунок' ✓

ТРЕБУЕТСЯ ОБНОВЛЕНИЕ: ДА - CSV должен иметь упрощённые названия без "Аэрография"

### 2️⃣ ДОПОЛНИТЕЛЬНЫЕ ЭЛЕМЕНТЫ (DecorativeElementsSelect)

КОМПОНЕНТ:
- Перья
- Имитация ремня с кнопками
- Бахрома
- Цветы
- Другое
- Ничего

CSV (текущие ключи):
- Декоративные элементы перья
- Декоративные  элементы имитация ремня с кнопками
- Декоративные элементы бахрома
- Декоративные элементы цветы
- Декоративные элементы другое

BACKEND маппинг (текущий):
  'feathers' → 'Декоративные элементы перья' ✓
  'belt' → 'Декоративные  элементы имитация ремня с кнопками' ✓
  'fringe' → 'Декоративные элементы бахрома' ✓
  'flowers' → 'Декоративные элементы цветы' ✓
  'other' → 'Декоративные элементы другое' ✓

ТРЕБУЕТСЯ ОБНОВЛЕНИЕ: НЕТ - маппинг правильный (CSV ключи соответствуют компоненту)

---

### 3️⃣ ДЕКОРАТИВНЫЙ ЭЛЕМЕНТ НА ПЛЕЧО (ShoulderStrapSelect)

КОМПОНЕНТ:
- Погон, украшенный камнями и/или бахромой
- Крылышко
- Подвесы (бретели, свисающие на предплечье)
- Без декоративного элемента на плечо

CSV (текущие ключи):
- Погон, украшенный камнями и/или бахромой ✓
- Крылышко ✓
- Подвесы

BACKEND маппинг (текущий):
  'decorated_shoulder' → 'Погон, украшенный камнями и/или бахромой' ✓
  'wing' → 'Крылышко' ✓
  'pendants' → 'Подвесы' ⚠️ (компонент: "Подвесы (бретели, свисающие на предплечье)")

ТРЕБУЕТСЯ ОБНОВЛЕНИЕ: ДА - CSV 'Подвесы' должен быть 'Подвесы (бретели, свисающие на предплечье)'

---

### 4️⃣ ТИП МОДЕЛИ КУПАЛЬНИКА (CombinaisionSelect)

КОМПОНЕНТ:
- Стандартный купальник
- Купальник на бретелях
- Комбинезон
- Комбинезон на бретелях

CSV (текущие ключи):
- Стандартный купальник ✓
- Купальник на беретелях (ОПЕЧАТКА: "беретелях" вместо "бретелях")
- Комбинезон ✓
- Комбинезон на бретелях ✓

BACKEND маппинг (текущий):
  'standard' → (базовая цена, плюс 0)
  'straps' → 'Купальник на беретелях' ⚠️
  'full' → 'Комбинезон' ✓
  'full_straps' → 'Комбинезон на бретелях' ✓

ТРЕБУЕТСЯ ОБНОВЛЕНИЕ: ДА - CSV 'Купальник на беретелях' → 'Купальник на бретелях'

---

  2. Плечо: исправить название
     - "Подвесы" → "Подвесы (бретели, свисающие на предплечье)"
  
  3. Комбинезон: исправить опечатку
     - "Купальник на беретелях" → "Купальник на бретелях"

После обновления CSV нужно обновить маппинги в backend (prices.js).
