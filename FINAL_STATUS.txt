## 📝 ТЕКУЩИЕ НАЗВАНИЯ В EXCEL

ЕСЛИ НУЖНО ОБНОВИТЬ, ТО НА:

1️⃣ АЭРОГРАФИЯ:
   Старые имена:          Новые имена:
   "Аэрография тонировка" → "Тонировка"
   "Аэрография переход..." → "Переход нескольких цветов"
   "Аэрография объемный..." → "Рисунок (персонаж, предмет, здание и т.д.)"

2️⃣ ПЛЕЧО:
   "Подвесы" → "Подвесы (бретели, свисающие на предплечье)"

3️⃣ КОМБИНЕЗОН:
   "Купальник на беретелях" → "Купальник на бретелях" (исправить опечатку)

## 🚀 КАК ИСПОЛЬЗОВАТЬ

1. Убедитесь, что Excel закрыт
2. Обновите CSV с новыми названиями (если хотите)
3. Запустите backend: cd backend && npm run dev
4. Запустите frontend: cd frontend && npm run dev
5. Откройте браузер: http://localhost:5173

## 📊 ЦЕНОВАЯ ЛОГИКА

✓ Базовая цена: Elite €500 / Standard €250 / Economy €150
✓ Модификаторы по росту: 170+ (20%), 150-170 (10%), 130-150 (5%), <130 (-5%)
✓ Множественные доп. услуги: суммируются
✓ Скидки: за отсутствие опций (-5%), за дизайн клиента (-20%)
✓ Срочность: стандарт (0%), ускоренный (10%), срочный (30-40%)
