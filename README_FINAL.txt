## 🚀 БЫСТРЫЙ СТАРТ

### Требования
- Node.js 14+
- npm 6+
- Браузер (Chrome, Firefox, Safari, Edge)

### Установка (1 раз)
```bash
# Backend
cd backend
npm install

# Frontend (в новом терминале)
cd frontend
npm install
```

### Запуск
```bash
# Терминал 1: Backend
cd backend
npm run dev
# Запустится на http://localhost:5001

# Терминал 2: Frontend
cd frontend
npm run dev
# Откройте http://localhost:5173
```

## 💰 ПРИМЕРЫ РАСЧЁТОВ

### Простой пример:
Elite, 175 см (рост 170+), 1 рукав, юбка спереди
= 500 + 100 (рост) + 15 + 20 = **635€**

### Сложный пример:
Elite, 175 см, 2 рукава, юбка спереди+сзади, перья+бахрома, 100 страз, срочно
= 500 + 100 + 30 + 30 + 20 + 100 + 40 (срочность) = **820€**

### С дисконтами:
Economy, 125 см, без рукавов, без юбки, "Ничего"
= 150 + 0 (рост) - 7.50 - 7.50 - 7.50 = **127.50€**

## 📞 ПОДДЕРЖКА И ОБНОВЛЕНИЯ

### Изменение цен
1. Отредактируйте файл `Prices.csv`
2. Перезапустите backend: `npm run dev`
3. Frontend автоматически подхватит новые цены

### Добавление новой опции
1. Добавьте строку в CSV
2. Создайте компонент в frontend
3. Обновите логику в prices.js
4. Протестируйте

### Вопросы?
Смотрите соответствующий файл документации:
- Как запустить? → PROJECT_COMPLETE.txt
- Как считаются цены? → TEST_CASES.txt
- Что изменилось в CSV? → XLSX_UPDATES_COMPLETE.txt
