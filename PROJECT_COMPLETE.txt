## 📐 ФУНКЦИОНАЛЬНОСТЬ

### Шаги конфигурации:

1️⃣ **Бюджет** (Budget Selector)
   - Выбор уровня: Elite (€500) / Standard (€250) / Economy (€150)

2️⃣ **Рост** (Height Slider)
   - Категории: <130, 130-150, 150-170, 170+
   - Модификаторы цены по росту

3️⃣ **Рукава** (Sleeves Select)
   - 0 рукавов (скидка -5%)
   - 1 рукав (+15€)
   - 2 рукава (+30€)

4️⃣ **Юбка** (Skirt Select)
   - Без юбки (скидка -5%)
   - Юбка спереди (+20€ Elite, +15€ Standard, +10€ Economy)
   - Юбка сзади (+10€ все уровни)
   - Юбка спереди и сзади (+30€ Elite, +25€ Standard, +20€ Economy)

5️⃣ **Декоративные элементы** (Decorative Elements)
   - Множественный выбор:
     • Перья
     • Имитация ремня с кнопками
     • Бахрома
     • Цветы
     • Другое
     • Ничего (скидка -5%)

6️⃣ **Декоративное плечо** (Shoulder Elements)
   - Множественный выбор:
     • Погон с камнями/бахромой (+15€)
     • Крылышко (+5€)
     • Подвесы (+5€)
     • Без декора (0€)

7️⃣ **Аэрография** (Aerography)
   - Множественный выбор:
     • Тонировка (+5€)
     • Переход цветов (+10€ Elite/Standard, +5€ Economy)
     • Рисунок персонажей (+10€)

8️⃣ **Тип модели** (Combinaison Type)
   - Стандартный купальник (0€)
   - Купальник на бретелях (+5€)
   - Комбинезон (+20% Elite/Standard, +15% Economy)
   - Комбинезон на бретелях (+20% Elite/Standard, +15% Economy)

9️⃣ **Премиум стразы** (Premium Stones)
   - Ss10-ss20: 100/200/300 штук (переменные цены)
   - Пришивные: 10/20/30 штук (переменные цены)

🔟 **Срочность** (Urgency)
   - Стандартный срок 7-14 дней (0%)
   - Ускоренный заказ 5 дней (+10%)
   - Срочный заказ 3 дня (+40% Elite, +30% Standard, +10% Economy)

1️⃣1️⃣ **Дизайн** (Design)
   - Наш дизайн от судьи (базовый)
   - Дизайн клиента (скидка -20%)

1️⃣2️⃣ **Пожелания** (Details Input)
   - Текстовое поле для дополнительных пожеланий

## 💰 ЦЕНОВАЯ ЛОГИКА

### Базовая цена (до модификаторов)
- Elite: €500
- Standard: €250
- Economy: €150

### Модификаторы по росту
- 170+: +20% (Elite/Standard), +15% (Economy)
- 150-170: +10%, +10%, +5%
- 130-150: +5%, +5%, 0%
- <130: -5%, -5%, -5% скидка

### Множественные опции (суммируются)
- Декоративные элементы: 5€-10€ за каждый
- Аэрография: 5€-10€ за каждый вариант
- Плечо: 5€-15€ за каждый элемент
- Стразы: 50€-200€ за количество

### Процентные надбавки
- Комбинезон: +15%-20% к цене с модификатором роста
- Срочность: +10%-40% к финальной цене

### Скидки (вычитаются)
- Отсутствие рукавов: -5% от базовой цены
- Отсутствие юбки: -5% от базовой цены
- "Ничего" (декор): -5% от базовой цены
- Дизайн клиента: -20% от базовой цены

## 📊 ПРИМЕРЫ РАСЧЁТОВ

### Пример 1: Базовый вариант
Elite, рост 175 см, 1 рукав, юбка спереди
→ 500€ + 100€ (рост) + 15€ + 20€ = **635€**

### Пример 2: Максимальный вариант
Elite, рост 175 см, 2 рукава, юбка спереди+сзади, все декоры, 2 вида стразов
→ 500€ + 100€ + 30€ + 30€ + 15€ + 15€ + 5€ + 5€ + 5€ + 100€ + 50€ = **855€**

### Пример 3: Минимальный вариант
Economy, рост 125 см, без опций
→ 150€ + 0€ (скидка за рост) - 7.5€ - 7.5€ - 7.5€ = **127.5€**

- Node.js 14+

**Backend:**

npm install
npm run dev
